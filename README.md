
# BITE-BURGER

Este es un menu de hamburgueseria, donde se podra ver las hamburgesas, precios y tiene conexion directa al WhatsApp de la hamburgueseria para poder realizar el pedido a gusto.


## Herramientas Utilizadas

 - HTML
 - CSS
 - REACT JS
 - BOOTSTRAP
 - FONT AWESOME
 - GOOGLE FONTS
 - TYPED JS
